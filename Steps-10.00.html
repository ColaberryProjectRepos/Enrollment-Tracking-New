Creating NoPayPer column using transform data.

  1. Click on Transform data
  2. Press and hold control button and select FullPayPer and PartialPayPer
  3. Go to Add Column and click on Standard and apply math operation Add
  4. New column is created name "Addition" and take that do the same steps and multiply with column name "TotalEnr" 
  5. New column is created name "Multiplication" and take that subtract from column name "TotalEnrl", make sure you select first the column name "TotalEnrl" when you do the subtraction to avoid get negative values.
  6. New column is created name "Subtraction"
  7. Take column name "Subtraction" and do the same steps and divide to column name "TotalEnrl" make sure the numerator will be the column name Subtraction and denominator will be the column name "TotalEnrl"
  8. New column is created name "Division" and change the name to "NoPayPer"
