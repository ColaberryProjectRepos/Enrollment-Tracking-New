Main KPI #1



Rev per Class


1.    Open Visualization Pane
2.    Select Visual Card
3.    Drag the visual to the dashboard
4.    Click on table Measure and select "Rev Per Class" measure, drag it into the fields.
5.    Visual: Callout value Color =#252423, Display units = Auto, Value Units = Auto, Text wrap = On, Source spacing = On, Category label = Off, Font type = DIN, size = 25,         Change format to currency  and decimal to 0.
6.    General: Properties, Height = 130, Width = 274, Position, Horizontal = 0, Vertical = 0, Title = On, Text Title = " Rev per Class", Font Family = DIN,  Font Size             = 15, Text Color = #000000, Background  Color = #A0D1FF,  Alignment = Center, Text wrap = On,
7.     Effects, Background = ON, Color = #FFFFF0, transparency = 0%, Visual border = Off, Shadow = Off, Header icon = Off, Tooltips =ON.
