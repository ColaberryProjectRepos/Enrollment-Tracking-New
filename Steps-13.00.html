Adding Project Title



1. Click on the Text box from Home or insert ribbon. Type the name of the project " Enrollment                  Tracking" 
2. Select the text then change the font size to 24 and font type Segoe UI. Select the text then                    alignment to center.
3. General: Properties, Height = 59, and Width = 766, Position, Horizontal = 274, Vertical = 0. Select          Effect then turn off the background to =0

Refreshed Date Setup




1.  Open Visualization Pane, Select Visual Card , Drag the visual to the dashboard. Click  on refreshed         date from Data Refresh Table and place it inside the fields. 
2.  General: Properties, Height = 51, Width = 240, Position Horizontal = 1040, Vertical =0
3.  Visual: Callout Value, Color = #000000, Font size 11, DIN, Category = Off
4.  General Title On, Text Title = " Date Refreshed ", Font Color = #252423, Background  Color no fill,         Font size = 11, Horizontal alignment to middle.
