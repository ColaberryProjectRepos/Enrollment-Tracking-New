Indicator #2

TotalEnrlPerInc indicator

1.    Open Visualization Pane

2.    Select Visual Card

3.    Drag the visual into the dashboard

4.    Click on” (TotalEnrPerInc Arrow) “and place it inside the fields

5.    General: Properties Height = 35, Width = 60, Position, Horizontal = 194, Vertical = 196, Header icons = Off

6.    Effects Background = On, Color = #FFFFF0

7.    Callout Font type DIN size = 18, Color click on function fx, Format style Rules, Fields        (Total Enrollments % Increase Vs.Prev Classes), Rules if value > Min and <= Max   then color #3CB11F, if value >= Min and < 0 then color  #EB1515, if value  =  0 then color #FFFFF0.
      Display units Auto, Value decimal places Auto, Text wrap On, Source spacing On Category label Off.

Total Enrollments % Increase Vs.Prev Class
Sub KPI #3

1.    Open Visualization Pane

2.    Select Visual Card

3.    Drag the visual into the dashboard

4.    Click on "(Total Enrollments % Increase Vs.Prev Class)" and place it inside the fields.

5.    General: Properties Height = 34, Width = 88, Position, Horizontal = 176, Vertical = 241, Header icons = Off, Tooltips = ON.

6.    Effects Background = On, Color = #FFFFF0

7.    Callout value, Font type "DIN", size = 18. Color = 000000, Click on function "Fx", Format style "Rules" Fields change to  " (Prj)", Rules if   value  >= Min  and  <= Max then color #000000, if value >=  Min and < 0 then color #000000, if  value = 0 then color #FFFFF0. Select measure (Total Enrollments % Increase Vs.PrevClass) then change the format to percentage and 1 decimal.

Proj_Total_Enroll
Sub KPI #4

1.    Open Visualization Pane

2.    Select Visual Card

3.    Drag the visual into the dashboard

4.    Click on "(Proj_total_enroll)" and place it inside the fields.

5.    General: Properties Height = 34, Width = 67, Position, Horizontal = 188, Vertical =153, Header icons = Off, Tooltips = ON.

6.    Effects Background = On, Color = #FFFFF0

7.    Callout value, Font type "DIN", size = 18. Color = 6A4023, Click on function "Fx", Format style "Rules" Fields change to " (Projection)", Rules if  value  >= Min and<= Max then color #6A4023, if value >=  Min and < 0 then color #6A4023, if  value = 0 then color #FFFFF0. Select measure (Prj) then change the format to whole number and 0 decimal.
